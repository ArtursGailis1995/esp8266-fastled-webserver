ESP8266 firmware update
The firmware must be updated by providing compiled *.bin file. When you click "Verify" button in Arduino IDE, *.bin file is compiled and stored in a %TEMP% folder, in folder starting with arduino_build_. You can use Win+R key combination and type in %TEMP% to quickly access temporary files location on your Windows computer.

The file system (SPIFFS) can be updated. To update it, click Tools > ESP8266 Sketch Data Upload in Arduino IDE. It will compile *.spiffs.bin file and place it in the same folder as *.bin firmware file. If it fails, make sure that ESP8266FS tool is installed and working, or update manually via USB.

ESP8266 must be power-cycled at least once after update before it can be updated again (disconnected from power source not restarted)!

To prevent losing SPIFFS contents after each update, make sure Tools > Erase Flash > Only Sketch is selected in Arduino IDE before compiling!
